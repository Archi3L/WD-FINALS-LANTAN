# 🎓 WD-FINALS-LANTAN – Personal Portfolio

A 5-page interactive and visually engaging personal website built for the Web Design & Client-Side Scripting finals. It combines dynamic layouts, canvas animations, and thematic sections—culminating with two standalone anime-inspired mini-sites.

---

## 🔍 Overview

This project reflects my personal and academic journey, presented in distinct, themed pages:

- **Landing Page** – Introduction with animated UI
- **About Me** – Chat-style intro with typing effects
- **Hobbies & Skills** – Console-style Q&A with effects
- **Gallery** – 3D project showcase (Blue Lock / Black Clover)
- **Thank You & Contact** – Final message with aurora-style animations

Each page features its own visual style, creating a cohesive yet varied user experience.

## ⚙️ Key Components

### 1. Animated Canvas Backgrounds
- **Landing Page**: Welcome effects  
- **About Me**: Typing chat bubbles  
- **Hobbies & Skills**: Matrix rain + terminal animation  
- **Gallery**: Rotating rings, sparkle & glowing dots  
- **Final Page**: Soothing aurora particles  

### 2. Interactive UI Elements
- Animated chat for introductions  
- Terminal Q&A experience  
- 3D hover effects on project cards  
- Page navigation buttons  
- Contact links with icons

### 3. Anime Project Mini-Sites
- **Blue Lock**: Anime character showcase  
- **Black Clover**: Multi-page character gallery  
  (Each built with its own styles, layouts, and assets)
